# My Profile App - Tugas 3 PAM

Aplikasi profil diri sederhana yang dibangun menggunakan **Jetpack Compose** (Kotlin). Aplikasi ini menampilkan informasi profil pengguna dengan antarmuka yang modern dan responsif.

## Fitur Utama
- **Profile Header**: Menampilkan foto profil (dalam bentuk circular/lingkaran) dengan background gradient yang menarik dan nama lengkap pengguna.
- **Bio/Deskripsi**: Ringkasan singkat mengenai identitas atau keahlian pengguna.
- **Detail Informasi**: Menggunakan komponen `Card` untuk merangkum informasi penting seperti:
  - **Email**: gilang.123140187@student.itera.ac.id
  - **Phone**: 085353341688
  - **Location**: Bandar Lampung, Indonesia
- **Reusable Components**: Kode disusun menggunakan fungsi `@Composable` yang dapat digunakan kembali seperti `ProfileHeader`, `InfoItem`, dan `ProfileCard`.

## Tech Stack
- **Language**: Kotlin
- **Framework**: Jetpack Compose
- **Theme**: Material Design 3 (Material3)
